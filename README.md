# Guess-The-Number
This is my 4th Macondo software project built using HTML, CSS, and JavaScript. Its just a Guess the Number Game.
# Languages
HTML: Used to build the structure of the game.
CSS: Used for styling the interface.
JavaScript: Used to handle the game's logic and functionality.
# Reason of Making
I just wanted to make this game as I saw many reels for it.
# Features
Only Integers input. Number less than 1 cant be entered. Number higher than 25 cant be entered. Tells whether Numbers is High or Low depending on the guess
